# 数据结构

## 栈（stack）

判断空栈：

```python
stack_empty(S)
	return S.top == 0
```

压栈：

```python
push(S, x)
	S.top++
	S[S.top] = x
```

弹栈：

```python
pop(S)
	if stack_empty(S)
		error "underflow"
	else
		S.top--
		return S[S.top + 1]
```

## 队列（queue）

入队：

```python
enqueue(Q, x)
	Q[Q.tail] = x
	if Q.tail == Q.length
		Q.tail = 1
	else
		Q.tail++
```

出队：

```python
dequeue(Q, x)
	x = Q[Q.head]
	if Q.head == Q.length
		Q.head = 1
	else
		Q.head++
	return x 
```

## 链表（linked list）

搜索：

```python
list_search(L, k)
	x = L.head
	while x != null and x.key != k
		x = x.next
	return x
```

插入：

```python
list_insert(L, k)
	x.next = L.head
	if L.head != null
		L.head.prev = x
	L.head = x
	x.prev = null
```

删除：

```python
list_delete(L, x)
	if x.prev != null
		x.prev.next = x.next
	else
		L.head = x.next
	if x.next != null
		x.next.prev = x.prev
```
