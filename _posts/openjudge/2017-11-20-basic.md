#### 上楼梯

问题描述

	上楼梯,可以一步上1级,可以一步上2级,也可以一步上3级,现在楼梯一共有N级,有多少种不同的上法
    
    (注意:当上到还剩下1级就到达时,不能直接上2级且不能直接上3级,当上到还剩下2级就到达时,不能直接上3级),

输入
    
    第一个数是M,表示M组测试数据,接下来有M行,每组测试数据一行,为一个N(1 <= N <= 20)
    
输出

    对于每一个测试数据,输出对应的不同的上楼梯方法

样例输入

    3
    
    1
    
    5
    
    10

样例输出

    1
    
    13
    
    274
    
思路

    n：台阶级数
    
    f(n)：走到第n级台阶时，剩余的不同走法
    
    1. 当n=1时，剩余走法f(1)=1种
    
    2. 当n=2时，剩余走法f(2)=2种
    
    3. 当n=3时，剩余走法f(3)=4种
    
如，当有4级台阶时，第一步可能跨出3种情况

    跨了1级，剩3级，有f(3)种走法
    
    跨了2级，剩2级，有f(2)种走法
    
    跨了3级，剩1级，有f(1)种走法
    
这些可能加起来就是4级台阶的不同上楼梯方法

总结：f(n) = f(n-3) + f(n-2) + f(n-1)
    
| n | f(n) | 可能 |
|:---:|:---:|:---:|
| 1 | 1 | 1 |
| 2 | 2 | 11, 2|
| 3 | 4 | 111, 12, 21, 3|
| 4 | 7 [f(3) + f(2) + f(1)] | ... |
| n | f(n-3) + f(n-2) + f(n-1)| ... |
    
代码

<pre class="highlight">
<code class="java">
import java.util.Scanner;

public class Main {
    public static void main(String args[]) {
        Scanner scanner = new Scanner(System.in);
        int m = scanner.nextInt();
        if (m > 0) {
            int[] array = new int[m];
            for (int i = 0; i < m; i++)
                array[i] = scanner.nextInt();
            for (int i = 0; i < m; i++)
                System.out.println(f(array[i]));
        } else {
            System.out.println(-1);
        }
        scanner.close();
    }
    // 递归
    // 数太大会爆
    private static int f(int n) {
        if (n == 1) {
            return 1;
        } else if (n == 2) {
            return 2;
        } else if (n == 3) {
            return 4;
        } else {
            return f(n - 3) + f(n - 2) + f(n - 1);
        }
    }
    
    // 迭代
    private static int fn(int n) {
        if (n == 1) {
            return 1;
        } else if (n == 2) {
            return 2;
        } else if (n == 3) {
            return 4;
        } else {
            int f1 = 1, f2 = 2, f3 = 4, fn = 0;
            for (int i = 3; i < n; i++) {
                fn = f1 + f2 + f3;
                f1 = f2;
                f2 = f3;
                f3 = fn;
            }
            return fn;
        }
    }
}

#### 重建二叉树

问题描述

    已知前序遍历和中序遍历，求后序遍历

输入
    
    输入可能有多组，以EOF结束
    
    每组输入包含两个字符串，分别为树的前序遍历和中序遍历
    
    每个字符串中只包含大写字母且互不重复
   
输出

    对于每组输入，用一行来输出它后序遍历结果
    
样例输入

    DBACEGF ABCDEFG
    
    BCAD CBAD

样例输出

    ACBFGED
    
    CDAB

代码

<pre class="highlight">
<code class="java">
import java.util.ArrayList;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        ArrayList<String> strings = new ArrayList<>();
        while (scanner.hasNext()) {
            String buf = scanner.next();
            if (buf.equals("EOF"))
                break;
            strings.add(buf);
        }
        scanner.close();
        char[] pre = null, in = null;
        for (int i = 0; i < strings.size(); i += 2) {
            pre = strings.get(i).toCharArray();
            in = strings.get(i + 1).toCharArray();
            TreeNode node = reConstruct(pre, 0, pre.length - 1, in, 0, in.length - 1);
            postOrder(node);
            System.out.println();
        }
    }

    public static class TreeNode {
        int value;
        TreeNode left;
        TreeNode right;

        TreeNode(int x) {
            value = x;
        }
    }

    public static TreeNode reConstruct(char[] pre, int preStart, int preEnd, char[] in, int inStart, int inEnd) {
        if (preStart > preEnd || inStart > inEnd)
            return null;
        TreeNode root = new TreeNode(pre[preStart]);
        for (int i = inStart; i <= inEnd; i++) {
            if (in[i] == pre[preStart]) {
                root.left = reConstruct(pre, preStart + 1, preStart + i - inStart, in, inStart, i - 1);
                root.right = reConstruct(pre, i - inStart + preStart + 1, preEnd, in, i + 1, inEnd);
            }
        }
        return root;
    }

    public static void preOrder(TreeNode node) {
        if (node != null) {
            System.out.print((char) node.value);
            preOrder(node.left);
            preOrder(node.right);
        }
    }

    public static void inOrder(TreeNode node) {
        if (node != null) {
            inOrder(node.left);
            System.out.print((char) node.value);
            inOrder(node.right);
        }
    }

    public static void postOrder(TreeNode node) {
        if (node != null) {
            postOrder(node.left);
            postOrder(node.right);
            System.out.print((char) node.value);
        }
    }
}

#### 青蛙的约会

问题描述

    两只青蛙在网上相识了，它们聊得很开心，于是觉得很有必要见一面。
    
    它们很高兴地发现它们住在同一条纬度线上，于是它们约定各自朝西跳，直到碰面为止。
    
    可是它们出发之前忘记了一件很重要的事情，既没有问清楚对方的特征，也没有约定见面的具体位置。
    
    不过青蛙们都是很乐观的，它们觉得只要一直朝着某个方向跳下去，总能碰到对方的。
    
    但是除非这两只青蛙在同一时间跳到同一点上，不然是永远都不可能碰面的。
    
    判断这两只青蛙是否能够碰面，会在什么时候碰面。
    
    规定纬度线上东经0度处为原点，由东往西为正方向，单位长度1米，这样就得到了一条首尾相接的数轴。
    
    设青蛙A的出发点坐标是x，青蛙B的出发点坐标是y。
    
    青蛙A一次能跳m米，青蛙B一次能跳n米，两只青蛙跳一次所花费的时间相同。
    
    纬度线总长L米。现在要你求出它们跳了几次以后才会碰面。

输入
    
    输入只包括一行5个整数 x，y，m，n，L，
    
    其中x≠y < 2000000000，0 < m、n < 2000000000，0 < L < 2100000000
   
输出

    输出碰面所需要的跳跃次数，如果永远不可能碰面则输出一行"Impossible"
    
样例输入

    1 2 3 4 5

样例输出

    4

代码

<pre class="highlight">
<code class="java">
import java.math.BigInteger;
import java.util.Scanner;

/**
 * 分析：典型的模线性方程求解
 *
 * ①已知n,m,L,x,y 设步数为 s，由题意得出方程：(ms-ns) % L = y-x 化简得：(m-n)s ≡ (y-x)(mod L)
 * ②根据扩展欧几里德算法求解模线性方程
 */
public class Main {

    private static BigInteger x;
    private static BigInteger y;

    // 扩展欧几里德算法
    public static BigInteger exGcd(BigInteger a, BigInteger b) {
        if (b.compareTo(BigInteger.valueOf(0)) == 0) {
            x = BigInteger.valueOf(1);
            y = BigInteger.valueOf(0);
            return a;
        } else {
            BigInteger d = exGcd(b, a.mod(b));
            BigInteger temp = x;
            x = y;
            y = temp.subtract(a.divide(b).multiply(y));
            return d;
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        BigInteger X = scanner.nextBigInteger();
        BigInteger Y = scanner.nextBigInteger();
        BigInteger M = scanner.nextBigInteger();
        BigInteger N = scanner.nextBigInteger();
        BigInteger L = scanner.nextBigInteger();
        BigInteger b = Y.subtract(X);
        BigInteger a = M.subtract(N);
        BigInteger n = L;
        BigInteger gcd = exGcd(a, n);
        if (b.mod(gcd).compareTo(BigInteger.valueOf(0)) == 0) {
            // 扩展欧几里德算法求解模线性方程中的 定理一
            x = x.multiply(b.divide(gcd)).mod(n);
            // 扩展欧几里德算法求解模线性方程中的 定理二
            // 得出的x可能是负数，需要求最小的正整数解，所以需要((x + n/d) % n/d
            x = x.add(n.divide(gcd)).mod(n.divide(gcd));
            System.out.println(x);
        } else {
            System.out.println("Impossible");
        }
        scanner.close();
    }
}

#### 整数奇偶排序

问题描述

    输入10个整数，彼此以空格分隔
    
    重新排序以后输出(也按空格分隔)，要求:
    
    1.先输出其中的奇数,并按从大到小排列
    
    2.然后输出其中的偶数,并按从小到大排列

输入
    
    任意排序的10个整数（0～100），彼此以空格分隔
   
输出

    按照要求排序后输出，由空格分隔
    
样例输入

    4 7 3 13 11 12 0 47 34 98

样例输出

    47 13 11 7 3 0 4 12 34 98
    
提示

    1. 测试数据可能有很多组，请使用while(cin>>a[0]>>a[1]>>...>>a[9])类似的做法来实现

    2. 输入数据随机，有可能相等

这题不难，但是输入那里有多组，每输入一组就输出对应结果

代码

<pre class="highlight">
<code class="java">
import java.util.Arrays;
import java.util.Scanner;

public class Main {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int[] array = new int[10];
        int[] after = new int[10];
        while (scanner.hasNext()) {
            for (int i = 0; i < 10; i++)
                array[i] = scanner.nextInt();
            Arrays.sort(array);
            for (int i = 9, evenIndex = 9, oddIndex = 0; i >= 0; i--) {
                if ((array[i] & 1) == 1) {
                    after[oddIndex] = array[i];
                    oddIndex++;
                } else {
                    after[evenIndex] = array[i];
                    evenIndex--;
                }
            }
            for (int i = 0; i < 10; i++) {
                System.out.print(after[i]);
                if (i != 9)
                    System.out.print(" ");
            }
            System.out.println();
        }
        scanner.close();
    }
}

#### 数字求和

问题描述

    给定一个正整数a，以及另外的5个正整数
    
    这5个整数中，小于a的整数的和是多少？

输入

    输入一行，只包括6个小于100的正整数，其中第一个正整数就是a
   
输出

    输出一行，给出一个正整数，是5个数中小于a的数的和
    
样例输入

    10 1 2 3 4 11

样例输出

    10

代码

<pre class="highlight">
<code class="java">
import java.util.Scanner;

public class Main {

	public static void main(String[] args) {
		Scanner scanner = new Scanner(System.in);
		int a = scanner.nextInt();
		int sum = 0;
		int buf = 0;
		for (int i = 0; i < 5; i++) {
			buf = scanner.nextInt();
			if (buf < a)
				sum += buf;
		}
		System.out.println(sum);
		scanner.close();
	}
}

#### 牛奶

问题描述

    计算采购足够数量的牛奶所需的最小花费。

    注：每天所有奶农的总产量大于Marry乳业的需求量

输入

    第 1 行共二个数值:N(0<=N<=2,000,000)是需要牛奶的总数；M(0<= M<=5,000)是提供牛奶的农民个数
    
    第 2 到 M+1 行：每行二个整数:Pi 和 Ai
    
    Pi(0<= Pi<=1,000) 是农民 i 的牛奶的价格
    
    Ai(0 <= Ai <= 2,000,000)是农民 i 一天能卖给Marry牛奶制造公司的牛奶数量
   
输出

    单独的一行包含单独的一个整数，表示Marry牛奶制造公司拿到所需的牛奶所要的最小费用
    
样例输入

    100 5
    
    5 20
    
    9 40
    
    3 10
    
    8 80
    
    6 30

样例输出

    630

代码

<pre class="highlight">
<code class="java">
import java.util.Collections;
import java.util.HashMap;
import java.util.Map;
import java.util.Scanner;

public class Main {

	public static void main(String[] args) {
		Scanner scanner = new Scanner(System.in);
		while (scanner.hasNext()) {
			int need = scanner.nextInt();
			int people = scanner.nextInt();
			Map<Integer, Integer> moneyAndNums = new HashMap<Integer, Integer>();
			int everyMoney = 0, totalNums = 0;
			for (int i = 0; i < people; i++) {
				everyMoney = scanner.nextInt();
				totalNums = scanner.nextInt();
                // hashmap的put方法会覆盖原有值
                // 如果刚好有同一价格的牛奶，应该加上原有数量
				if (moneyAndNums.get(everyMoney) != null)
					totalNums += moneyAndNums.get(everyMoney);
				moneyAndNums.put(everyMoney, totalNums);
			}
			int spare = 0;
			while (need != 0) {
				everyMoney = Collections.min(moneyAndNums.keySet());
				totalNums = moneyAndNums.get(everyMoney);
                // 如果牛奶数量不够多就全买了
                // 如果牛奶数量比需求多，那需求已经满足了
				if (need < totalNums)
					totalNums = need;
				spare += everyMoney * totalNums;
				need -= totalNums;
				moneyAndNums.remove(everyMoney);
			}
			System.out.println(spare);
		}
		scanner.close();
	}
}

#### 最大乘积

问题描述

    给定一个整数数组，我们希望去掉其中一个数，使得剩下所有数的乘积最大，问应该去掉哪一个数

输入

    第一行M，表示有M组数据
    接下来2*M行
    每两行表示一组测试数据
    每组的第一行是一个整数N，表示是数组的长度
    接下来一行有N个整数，表示数组的内容，3<=N<=100，且数组内每个数都在[-10000000,10000000]区间内
   
输出

    M行，每行一个整数，表示应该去掉的那个数，如果有多个选择，则输出最先输入的那个
    
样例输入

    4
    3
    0 1 2
    5
    2 3 5 4 8
    5
    -1 -2 -3 -4 -5
    4
    -1 -2 -3 -4

样例输出

    0
    2
    -1
    -4

分析
    
    1.乘积为0：
    ____1.1 数组中有多个0：去掉第一个数；
    ____1.2 数组中有且仅有一个0：
    ________1.2.1 其余数的乘积为正数：去掉0；
    ________1.2.2 其余数的乘积为负数：
    ______________1.2.2.1 数组第一个数为0，去掉第二个数；
    ______________1.2.2.2 数组第一个数非0，去掉第一个数。
    ______________（意思就是去掉第一个非0的数）

    2.乘积为正：
    2.1 数组完全由偶数个负数组成，没有正数：去掉最小的负数（绝对值最大）；
    2.2 数组中有正数，说明其中还有偶数个负数：去掉最小的正数。

    3.乘积为负：
    这种情况很简单，去掉最大的负数（绝对值最小）。

import java.util.Scanner;

public class Main {
	public static void main(String args[]) {
		Scanner scanner = new Scanner(System.in);
		int n = scanner.nextInt();
		int m, i, j;
		int curStartIndex = 1, curSum = 0;
		int startIndex = 0, endIndex = 0, maxSum = 0;
		int tmp = 0;
		for (i = 0; i < n; i++) {
			m = scanner.nextInt();
			curStartIndex = 1;

			maxSum = 0;

					curSum = tmp;
					curStartIndex = j + 1;
				} else
					curSum += tmp;
				if (maxSum < curSum) {
					maxSum = curSum;
					startIndex = curStartIndex;
					endIndex = j + 1;
				}
			}
			System.out.printf("Case %d:\n", i + 1);
			System.out.printf("%d %d %d\n", maxSum, startIndex, endIndex);
			if (i != n - 1)
				System.out.println();
		}
