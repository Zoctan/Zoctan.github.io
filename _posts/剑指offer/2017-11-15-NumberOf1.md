#### 介绍

题目描述

	输入一个整数，输出该数二进制表示中1的个数。
	
	其中负数用补码表示

方法1

有多少个1，就去掉多少个1。

利用减1后的数和原数相与，循环去掉最后的1。

如：十进制的12，二进制为1100，12-1后变为1011，原来的1100和1011相与：1000，即去掉了最右边的1，循环直到数为0。

<pre class="highlight">
<code class="java">
public int NumberOf1(int n) {
    int nums = 0;
    while (n != 0) {
        n &= n - 1;
        nums++;
    }
    return nums;
}
</code>
</pre>

方法2

循环左移，&2的次方判断（从2的0次方开始，即8 4 2 1）

如：十进制11，二进制为1011，&1判断第1位后，将1左移<<变为10，判断第2位，循环直到2的次方等于0。

（int i左移溢出变回0，可以判断32次）

<pre class="highlight">
<code class="java">
public int NumberOf1(int n) {
    int nums = 0;
    int i = 1;
    while (i != 0) {
        if ((n & i) == i)
            nums++;
        i <<= 1;
    }
    return nums;
}
</code>
</pre>

方法3

java有逻辑右移，可以忽略负数补1的情况

<pre class="highlight">
<code class="java">
public int NumberOf1(int n) {
    int nums = 0;
    while (n != 0) {
        if ((n & 1) == 1)
            nums++;
        n >>>= 1;
    }
    return nums;
}
</code>
</pre>
