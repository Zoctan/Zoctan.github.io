树状数组

树状数组/二叉索引树（Binary Indexed Tree）：

A 数组就是原数组，C 数组则是树状数组。

通过观察可以发现：

它的查询和修改的时间复杂度都是 O(logN)，空间复杂度则为 O(N)，这是因为树状数组通过将线性结构转化成树状结构，从而进行跳跃式扫描。

通常使用在高效的计算数列的前缀和，区间和。
