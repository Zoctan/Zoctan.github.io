    原根

    2018-03-14
    • 算法
    word   511 min2read   3   minute  

    http://www.51nod.com/onlineJudge/questionCode.html#!problemId=1135

    设m是正整数，a是整数，若a模m的阶等于φ(m)，则称a为模m的一个原根。（其中φ(m)表示m的欧拉函数）
    给出1个质数P，找出P最小的原根。

    Input

    输入1个质数P(3 <= P <= 10^9)

    Output

    输出P最小的原根。

    Input示例

    3

    Output示例

    2
