给出1个正整数N，检测N是否为质数。如果是，输出”Yes”，否则输出”No”。

Input

输入一个数N(2 <= N <= 10^30)

Output

如果N为质数，输出”Yes”，否则输出”No”。

Input示例

17

Output示例

Yes
