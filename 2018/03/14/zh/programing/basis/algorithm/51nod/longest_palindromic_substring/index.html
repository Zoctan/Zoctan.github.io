最长回文子串

最长回文子串

http://www.51nod.com/onlineJudge/questionCode.html#!problemId=1088

回文串是指aba、abba、cccbccc、aaaa这种左右对称的字符串。
输入一个字符串Str，输出Str里最长回文子串的长度。

Input

输入Str（Str的长度 <= 1000)

Output

输出最长回文子串的长度L。

Input示例

daabaac

Output示例

5
