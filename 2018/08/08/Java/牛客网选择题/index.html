    牛客网选择题

    2018-08-08
    • Java
    字数   98 预计阅读   1   分钟  

    Java 的屏幕坐标是以像素为单位，容器的左上角被确定为坐标的起点。

    初始化执行顺序：

    • 静态初始化块 > 初始化块 > 构造器
    • 父类 > 子类

    综合下来顺序就是：

    • 父类静态初始化块
    • 子类静态初始化块
    • 父类初始化块
    • 父类构造器
    • 子类初始化块
    • 子类构造器
