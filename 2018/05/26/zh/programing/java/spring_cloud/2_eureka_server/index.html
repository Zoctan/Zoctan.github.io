  • 一个服务注册中心 Eureka Server 端口为7000

  • Consumer 端口为9000，向服务注册中心注册
  • 当 Consumer 通过 RestTemplate 调用 Client 的 add 接口时，因为用 Ribbon 进行了负载均衡，会轮流的调用 Client：8000、8001 两个端口的 add 接口
