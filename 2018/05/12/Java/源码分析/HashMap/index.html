    特点

    1. HashMap 是一个散列表，它存储的内容是键值对（key-value）映射。
    2. 允许存在一个 key 为 null 和任意个 value 为 null 的键值对。
    3. HashMap 继承于 AbstractMap，实现了 Map、Cloneable、java.io.Serializable 接口。
    4. HashMap 的方法没有锁，即其不是线程安全的。
