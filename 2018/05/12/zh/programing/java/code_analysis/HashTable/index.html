特点

  1. Hashtable 是一个散列表，它存储的内容是键值对（key-value）映射。
  2. key、value 都不可以为 null。
  3. Hashtable 继承于 Dictionary，实现了 Map、Cloneable、java.io.Serializable 接口。
  4. Hashtable 的方法都是同步（synchronized）的，即其是线程安全的。
